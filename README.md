# 🎧 每日音乐电台

个人音乐推荐系统。每天从全球独立音乐中挖掘新歌，用 YouTube 内嵌播放器直接听。

**在线地址：https://redleafluan.github.io/music-daily/**

---

## 怎么用

### 听歌

手机浏览器打开上面的链接：

| 按钮 | 效果 |
|------|------|
| ▶ 全部播放 | 从第 1 首开始，底部弹出 YouTube 播放器 |
| 🎲 随机 | 随机抽一首播 |
| ☆ | 收藏（页面刷新后保留） |
| ◀ ▶ | 切换往期推荐 |
| ✏️ | 标记这首歌要写小红书文案，底部汇总「待写文案」列表 |

### 写文案

点 ✏️ 标记的歌会出现在页面底部「📝 待写文案」区域。写完点 ✓ 标记完成。

文案素材在 `xiaohongshu-issue*.md` 文件里，每首歌都有事实查证过的创作背景 + 推荐文案，直接复制发小红书。

### 给我反馈

在 Claude Code 对话框里告诉我：
> 「歌名 — 喜欢/一般/不喜欢 — 简单一句话」

我会更新你的味觉地图，下次推荐更准。

---

## 每天怎么运行

1. **打开 Claude Code**（App 或终端都可以）
2. 等它自动触发每日推荐任务（每天 8:23）
3. 或者手动说：**「跑每日音乐推荐」**
4. 推荐会在对话框中显示，同时自动推送到 GitHub
5. 手机刷新 **https://redleafluan.github.io/music-daily/** 就有新歌

## 搜索策略（双轨）

| | 轨1 · 国内有源 | 轨2 · 全球 |
|------|:--:|:--:|
| 来源 | 网易云、QQ音乐、B站 | YouTube、Bandcamp、RYM、Spotify |
| 能分享 | ✅ 小红书/微信 | ❌ 只自己听 |
| 每期 | 2-3 首 | 2-3 首 |

筛选标准基于个人「音乐味觉地图」（存储在 Claude Code 记忆中）：层层堆叠的情绪叙事、空灵女声、旋律驱动、不炸不吵。
